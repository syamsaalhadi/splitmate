# SplitMate 💸
### Aplikasi Manajemen Patungan & Utang Cerdas untuk Gen Z

SplitMate adalah aplikasi web fintech yang membantu generasi muda mengelola 
patungan, utang piutang antar teman, dan pengeluaran bersama secara lebih 
terstruktur, transparan, dan cerdas menggunakan kecerdasan buatan.

## 🚀 Fitur Utama
- **Split Otomatis** — Hitung pembagian biaya sama rata atau custom per orang
- **Tracking Utang** — Pantau utang & piutang real-time antar anggota grup
- **AI Insight** — Klasifikasi kategori pengeluaran & prediksi pengeluaran bulan depan
- **Dashboard Keuangan** — Visualisasi pola pengeluaran dan skor finansial personal

## 🛠️ Tech Stack
| Layer | Teknologi |
|-------|-----------|
| Frontend | React + Vite |
| Backend | FastAPI (Python) |
| Database | PostgreSQL (Supabase) |
| AI / ML | TensorFlow / Keras |
| Data Science | Streamlit + Pandas |
| Deployment | Netlify / Vercel + Streamlit Cloud |
